Anagram

This is a tool made to create anagrams. Once the characters are generated from the initial word, they can be locked by clicking on them. The Randomize button will shuffle the characters which are not locked.

Flexbox

A CSS sandbox showing what the flex display behavior can do.

Gridmoves

This page creates a grid. Each cell on the grid is red if it is close enough to the center, and gray otherwise. The user can choose which distance is used : manhattan, eucledian, or minkowski.

Passwords

A simple passeword generator, which use the built-in Math.random() javascript function.

Radima

Radima is an Amidar clone with an Antartic theme. The goal of the game is to collect all the pellets to clear the rectangles. Clearing the four corners will make the player able to eat the enemies for a while. The player can press space to freeze the enemies.
